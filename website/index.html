First open database for energy data

We are creating is the first oopen distributed database for smart meter data. We started this project end of August 2018. In this first phase we will focus on onboarding smart meters, so the data starts populating. In phase two we will create a data explorer for everyone to use. If the basic data explorer is there there will be data analysis tools.

Goal of this project is to make energy data more accessible, so nice analysis can be applied. Right now data is collected in private silo's. We as humankind can innovate faster if the anonymized data is publishly available. By developing open source software and publishing data onto a open, distributed database, we will give the data back to the people. Analysis tools follow.

Want to join the mission? There are two ways to participate:

  1. Sign up your smart meter!

    We provide you documentation on how to start sharing your smart meter energy data. You'll need a Raspberry Pi and the VEH open source code, and you're ready to go!



  2. Sign up for the newsletter

    Curious to the developments of this project. Want to get a notification when the first data streams are live? Sign up for our newsletter & we will keep you up to date!
